# Especificaciones técnicas y utilidad de los códigos por categoría

## ingenieria/bombas_y_flujos/
- **CURVA_BOMBA.py, CURVA_BOMBA_2.py, CURVA2.py, curva_mtto.py, sfd1.py**
  - **Propósito:** Cálculo y graficación de curvas de operación de bombas y sistemas de tuberías. Análisis de esfuerzos y caudales.
  - **Aplicación:** Selección, diseño y mantenimiento de equipos hidráulicos en ingeniería mecánica.
  - **Dependencias:** matplotlib, numpy
  - **Ejemplo de uso:** `python CURVA_BOMBA.py`

## ingenieria/metodos_numericos/
- **RUNGE_KUTTA.py**
  - **Propósito:** Resolución de ecuaciones diferenciales mediante el método de Runge-Kutta.
  - **Aplicación:** Simulación de sistemas dinámicos y procesos físicos.
  - **Dependencias:** numpy
  - **Ejemplo de uso:** `python RUNGE_KUTTA.py`

## ingenieria/visualizacion/
- **Grafica_Duran.py, Grafica_duran_1.py, Graph_durant.py**
  - **Propósito:** Graficar resultados de simulaciones, tendencias y análisis de datos experimentales.
  - **Aplicación:** Visualización de datos en proyectos de ingeniería.
  - **Dependencias:** matplotlib
  - **Ejemplo de uso:** `python Grafica_Duran.py`

## analisis/financiero_operativo/
- **Analisis.py, Analisis_1.py, Opex_2025.py, Requerimiento_mes.py**
  - **Propósito:** Análisis de costos, proyecciones y requerimientos de recursos.
  - **Aplicación:** Evaluación económica y financiera de proyectos de ingeniería.
  - **Dependencias:** pandas, numpy
  - **Ejemplo de uso:** `python Opex_2025.py`

## estadistica/analisis_datos/
- **ESTADISTICA.py**
  - **Propósito:** Análisis estadístico de datos de operación, fallas y mantenimiento.
  - **Aplicación:** Toma de decisiones basada en datos en ingeniería.
  - **Dependencias:** pandas, numpy, matplotlib
  - **Ejemplo de uso:** `python ESTADISTICA.py`

## mantenimiento/gestion/
- **gestion_mtto.py**
  - **Propósito:** Automatización y control de actividades de mantenimiento preventivo y correctivo.
  - **Aplicación:** Gestión eficiente de recursos y equipos en plantas industriales.
  - **Dependencias:** pandas, sqlite3
  - **Ejemplo de uso:** `python gestion_mtto.py`

## gui/simuladores/
- **MODELO_GRUA.py**
  - **Propósito:** Simulación y estudio de planes de izaje seguro con grúas, integrando cálculos, visualización y cuestionarios.
  - **Aplicación:** Capacitación, análisis de seguridad y planeación de operaciones de izaje.
  - **Dependencias:** tkinter, matplotlib
  - **Ejemplo de uso:** `python MODELO_GRUA.py`

## scripts_utilidad/conversiones/
- **from_kivy.py, p6_1.py, sfd1.py, curva_mtto.py, CURVA2.py, CURVA_BOMBA.py, CURVA_BOMBA_2.py**
  - **Propósito:** Scripts de apoyo para cálculos, conversiones y automatización de tareas repetitivas.
  - **Aplicación:** Utilidades generales para ingenieros y técnicos.
  - **Dependencias:** variable según script
  - **Ejemplo de uso:** `python nombre_script.py`
